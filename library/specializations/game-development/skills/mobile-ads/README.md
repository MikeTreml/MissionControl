# Mobile Ads Skill

Implement and optimize ads with a player-first monetization strategy.

## What this skill covers

- rewarded, interstitial, and banner placement patterns
- mediation network setup and waterfall/bidding considerations
- consent flows and privacy-safe initialization
- pacing controls (frequency caps, cooldowns, session rules)
- instrumentation and experiment design

## Typical workflow

1. Define monetization goals and guardrails (retention, session length).
2. Configure placements and mediation defaults per platform.
3. Add analytics for request, fill, impression, click, reward, and error.
4. Tune placement timing and caps using experiment cohorts.
5. Review revenue lift versus player experience regressions.

## Deliverables

- integration checklist and rollout notes
- placement matrix with rationale
- metrics dashboard query list
- optimization backlog for next iteration
