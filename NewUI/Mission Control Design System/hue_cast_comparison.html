Pure neutral reads clinical / Apple-modern. No personality, gets out of the way. Good for content-heavy tools where the UI shouldn't compete with the work.

Warm cast reads cozy / premium / hand-crafted. Linear and Cron use this — it makes a tool feel less like infrastructure and more like a workshop. Recommended for Mission Control given the orchestration / craft framing.

Cool cast reads technical / sober. Different from the AI-cliché blue-black because it's 3% chroma, not 30% — but at scale it can still feel a touch cold. Safe choice for an engineering tool.
